## Build Setup

``` bash
# git clone
git clone https://github.com/RowinRuizendaal/project-tech.git

# Cd into project
cd project-tech

# Install dependencies 
npm install

# serve with hot reload at localhost:3000
Nodemon index.js || npx nodemon index.js
```
